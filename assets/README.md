# Project assets
This project uses the NASA SRTMGL1 1-arcsecond global DEM for heightmap data.

## Ex. SRTM 1-arcsecond tile N40W106
The 1-arccsecond DEMs are available from NASA's public data archive. Here's the best one:
```bash
cd assets
wget https://step.esa.int/auxdata/dem/SRTMGL1/N40W106.SRTMGL1.hgt.zip
unzip N40W106.SRTMGL1.hgt.zip
```
